Food Recognition & Calorie Estimation Web App

This project is a **web-based application** that recognizes food items from images and estimates their **calorie content**.  
It helps users track their daily food intake and make **health-aware dietary choices** through an interactive and user-friendly interface.

Task Description
Develop a web application to recognize food items from images and estimate calorie content using front-end technologies.

Project Objectives
- Identify food items from uploaded images  
- Display estimated calorie values  
- Help users monitor daily calorie intake  
- Provide a clean and interactive UI  

Technologies Used
- HTML  
- CSS  
- JavaScript  

Dataset Reference
- Kaggle – Food-101 Dataset  
- Link: https://www.kaggle.com/datasets/dansbecker/food-101

> Note: Dataset is used as a reference for food categories and calorie estimation logic.

Files in this Repository
- `index.html` – Main web page  
- `style.css` – Styling and UI design  
- `script.js` – Application logic and interaction handling  
- `README.md` – Project documentation  

Application Features
- Upload food image  
- Analyze food item  
- Display calorie estimation  
- Daily calorie tracking summary  
- Interactive and responsive UI
